Definição recursiva III

Seja uma função recursiva C tem dois argumentos e devolve verdade se existe caminho entre elas.

$$ C(f, d) = \begin{align} \begin{cases} true & se \\ true & se \\[20pt] false & c.c \end{cases} \end{align} $$
